# Projeto Perfil Dev

Este projeto foi desenvolvido como parte da atividade prática de **versionamento com Git e GitHub**.  
O objetivo é demonstrar na prática o uso de controle de versões em um projeto simples de perfil de desenvolvedor.

Durante o desenvolvimento foram aplicados conceitos fundamentais de Git como:

- Inicialização de repositório
- Versionamento de arquivos
- Criação de commits
- Criação e utilização de branches
- Merge de branches
- Integração com repositório remoto no GitHub

Essas práticas são essenciais no desenvolvimento de software moderno, permitindo organização do código, rastreamento de alterações e colaboração entre desenvolvedores.

---
